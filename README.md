# ARKA Protocol Plugins

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

Open source reference plugin implementations for ARKA Protocol.

## Available Plugins

| Plugin | Description |
|--------|-------------|
| `@arka/aml` | Anti-Money Laundering compliance plugin |
| `@arka/audit-log` | Audit logging and trail management |

## Installation

```bash
npm install @arka/aml
# or
npm install @arka/audit-log
```

## Creating Your Own Plugin

See the [Plugin Development Guide](https://github.com/arka-protocol/arka-core) for documentation on building custom plugins.
